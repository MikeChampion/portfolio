Note Taker

An app that keeps track of all your todo notes. Add new notes (with a title, and a longer description) or review existing notes.

This project utilizes HTML, CSS, and JavaScript for the front-end.
The back-end uses Express for the server, and UUID for each note's unique ID.
It is deployed using Heroku.
